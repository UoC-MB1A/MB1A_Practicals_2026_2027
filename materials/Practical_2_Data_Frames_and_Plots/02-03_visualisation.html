    Initiating a new plot

    ↑ top

    We will visualise the spring migration paths of the birds using the R plotting package ggplot.

    We can do this in the same Quarto document used for Part 1.

    ggplot builds figures by starting with a blank plot and then adding layers. We first tell ggplot which data frame to use, then specify the variables to plot, and finally add layers such as points, lines and axes.

    As we saw in Practical 1 (Plotting in R), we first need to generate a new blank plot.

    We do this using the function ggplot(), with our data frame as an argument.

    ggplot(spring_data)

    Based on the published plot, we know that our x axis should be Longitude and our y axis should be Latitude.

    Adding points

    ↑ top

    A plot of longitude against latitude is simply a scatter plot - longitude determines the position on the x axis and latitude determines the position on the y axis.

    We can add points to our plot using the geom_point() function, just like we used geom_line() to add a line in the previous practical (Plotting in R).

    To demonstrate the geom_point() function, let’s briefly revisit our fruit data frame from the previous practical.

    my_data_frame <- tibble(
  fruit = c("apple", "pear", "banana"),
  length = c(22, 30, 55),
  weight = c(150, 165, 170)
)

gt(my_data_frame)
    fruit length weight
    apple 22 150
    pear 30 165
    banana 55 170

    To plot this data as a scatter plot, with length on the x axis and weight on the y axis, we would do this:

    ggplot(my_data_frame, aes(x = length, y = weight)) +
  geom_point()
    ExerciseExercise 2

    Going back to your worked example Quarto document, add a layer to your existing plot to plot all of the longitude and latitude points as a scatter plot.

    AnswerAnswer 
    ggplot(
  spring_data,
  aes(x = Longitude, y = Latitude)
) +
  geom_point()

    If you compare this with Fig 1B in the paper, you can see that the points are at the same positions.

    Now, we can add some additional information to make the plot more informative.

    Adding map features

    ggplot has a convenient function annotation_borders() to add geographical country borders to a scatter plot representing a map. We’re also going to add a second function used with map data, coord_quickmap() - this keeps the shape of the plot consistent so the map isn’t skewed.

    We can add this layer to our scatter plot as follows. The argument linewidth sets the width of the lines in the map borders.

    Important

    The majority of the steps in this practical are useful for any scatter plot, however, this specific code chunk is specific to plotting maps.

    NoteInclude this code

    In your worked example Quarto document, add this layer (starting at annotation_borders) to your existing scatterplot to add map borders.

    ggplot(
  spring_data,
  aes(x = Longitude, y = Latitude)
) +
  geom_point() +
  annotation_borders(linewidth = 0.2) +
  coord_quickmap()

    Setting axis limits

    ↑ top

    It would be clearer to zoom in just on the region containing the points, we can do this by setting the axis limits.

    It would be helpful to know the minimum and maximum longitude and latitude of the points, we can do this with the min() and max() functions and the function pull(), which pulls a single column from a data frame.

    e.g. to find the minimum longitude in our data frame

    min_long <- spring_data |>
  pull(Longitude) |>
  min()

print(min_long)
    [1] -13.54433

    We can use these numbers directly as our axis limits - but let’s subtract 2 from each minimum and add 2 to each maximum so our points are not right on the edge of the plot.

    We can set the x axis limits as an argument to the coord_quickmap() function as follows:

    Important

    For non-map data, we would most commonly use coord_cartesian() instead of coord_quickmap() here, which works in the same way.

    ggplot(
  spring_data,
  aes(x = Longitude, y = Latitude)
) +
  geom_point() +
  annotation_borders(linewidth = 0.2) +
  coord_quickmap(xlim = c(min_lon - 2, max_lon + 2))
    ExerciseExercise 4

    ylim = c(min_lat - 2, max_lat + 2)

    We want to set the colour by the Site column and the shape by the Location_Type column.

    To set colour by Site, we add color = Site as an argument to the aes() function.

    ggplot(
  spring_data,
  aes(x = Longitude, y = Latitude, color = Site)
) +
  geom_point() +
  annotation_borders(linewidth = 0.2) +
  coord_quickmap(
    xlim = c(min_lon - 2, max_lon + 2),
    ylim = c(min_lat - 2, max_lat + 2)
  )

    Setting shape by Location_Type is very similar, we add shape = Location_Type as an argument to the aes() function.

    Adding paths

    ↑ top

    Finally, we want to add paths to join the points from the same location.

    To do this, we use the geom_path() function, grouping the points by breeding site, by adding the argument aes(group = Site):

    ggplot(
  spring_data,
  aes(x = Longitude, y = Latitude, color = Site, shape = Location_Type)
) +
  geom_point() +
  annotation_borders(linewidth = 0.2) +
  coord_quickmap(
    xlim = c(min_lon - 2, max_lon + 2),
    ylim = c(min_lat - 2, max_lat + 2)
  ) +
  geom_path(aes(group = Site))

    There are various tweaks available if you want to change the appearance of your plot.

    For example:

    • To change the size of the points, add e.g. size = 3 as an argument to the geom_point() function.
    • To change the width of the line, add e.g. linewidth = 0.5 as an argument to the geom_line() function.
    • To change the colour theme of all the data points, add an additional layer with scale_color_brewer(palette = “PALETTE”) , changing PALETTE to anything on the chart on this page.
    • You can change the overall theme of your plot using one of the theme options on this page
    • You can add a title by adding a layer ggtitle(), with the title as the only argument.

    It’s worthwhile to search online for other aesthetic changes you can make to your plot, many options are available.

    Add any of these that you like to the plotting code in your worked example Quarto document.

    E.g.

    ggplot(
  spring_data,
  aes(x = Longitude, y = Latitude, color = Site, shape = Location_Type)
) +
  geom_point(size = 4) +
  annotation_borders(linewidth = 0.2) +
  coord_quickmap(
    xlim = c(min_lon - 2, max_lon + 2),
    ylim = c(min_lat - 2, max_lat + 2)
  ) +
  geom_path(aes(group = Site), size = 1) +
  scale_color_brewer(palette = "Dark2") +
  theme_bw() +
  ggtitle("Spring Migration Routes")
    Warning: Using `size` aesthetic for lines was deprecated in ggplot2 3.4.0.
ℹ Please use `linewidth` instead.
